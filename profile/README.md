## 🎯 Missiya

HackNow — bu O'zbekistondagi eng yirik kiberxavfsizlik ekotizimi. Biz nazariya va amaliyot orasidagi masofani yo'qotish va har bir foydalanuvchini real kiber-tahdidlarga qarshi tura oladigan darajaga yetkazishni maqsad qilganmiz.

## 🚀 Loyihalar

| Loyiha | Tavsif |
|--------|--------|
| **HackNow Platform** | Asosiy ta'lim platformasi — CTF, CP, darsliklar, reyting tizimi |
| **HackNow Landing** | Landing sahifa — hacknow.uz |
| **Attack Framework** | Red Team va penetration testing vositalari |
| **Machine Lab** | Virtual mashinalar bilan ishlash laboratoriyasi |
| **Web Lab Runner** | Web-based CTF challenge runner |
| **HackNow CLI** | Terminal-based platforma interfeysi |
| **LunarN** | AI-powered kiberxavfsizlik yordamchisi |
